# L3 схема сети и ip-план

Должна включать VLAN, их номера, логические (ip) адреса  ключевых устройств
- Сегменты сети
- Маршрутизирующие узлы
- Ключевые устройства

Не отражает:
- Физические порты
- Оборудование, которое работает на L2
